Règles

But du jeu

Ce jeu consiste à construire des phrases avec d'autres joueurs, tout en ignorant les mots entrés par ceux-ci.
 La phrase ainsi créée est absurde.
Il est inspiré de la version papier et manuscrite du jeu original.

Comment jouer

Pour participer, il faut créer une nouvelle phrase, ou compléter une existante.
 Une phrase sera toujours construite de la même manière : Un sujet, un verbe, un complément d'objet direct et un complément circonstanciel.

Créer une phrase :

Pour créer une phrase, se rendre sur le module Phrase en cours et cliquer sur Créer (vous devez être identifié).
 Entrez un sujet en faisant preuve d'imagination, et validez votre mot.
 Ainsi vous initialisez une nouvelle phrase, qui apparaitra cachée dans le module Phrases en cours.
 D'autres joueurs pourront ainsi la compléter, et elle se dévoilera dans le module Phrases terminées une fois qu'elle sera complète.

Compléter une phrase

Pour compléter une phrase, se rendre dans le module Phrases en cours et sélectionner une des phrases.
 Vous êtes alors invités à y ajouter un mot - ou suite de mots (vous devez être identifié).
 Si vous la finissez, celle-ci apparaitra dans le module Phrases terminées.

Bonus : générer une phrase aléatoirement

Dans le module Générer une phrase aléatoirement, cliquer sur Générer une phrase pour créée une phrase aléatoirement à partir des mots entrés par les joueurs.
